# nlw expert - trilha Java

## certification_nlw
Projeto para aplicação e realização de provas com certificado.
- DTO;
- Banco de dados Postgres;
- Persistência dos dados com Spring Data JPA e Hibernate ORM;
- Docker Compose;
- Princípios de SOLID;
- Lambda.

---------------------------------------------------------

## Using
- Maven
- SpringBoot
- Docker
- Java
- VSCode
